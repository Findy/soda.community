# SODA Community

SODA Community は、AI時代のソフトウェアエンジニアリングを探求し、
知識を共有するためのオープンコミュニティです。

このリポジトリではコミュニティ運営（合意形成・方針・イベント等）を扱います。
公開ナレッジ（記事・図・表）は別リポジトリ **soda.dev** で管理・公開します。
